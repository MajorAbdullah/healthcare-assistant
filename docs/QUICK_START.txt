┌─────────────────────────── QUICK BOOKING WORKFLOW ───────────────────────┐
│                                                                           │
│  Step 1:  Select Menu Option [2] → Book an appointment                   │
│  Step 2:  Choose Doctor          → Select ID (1, 2, or 3)                │
│  Step 3:  Pick Date               → YYYY-MM-DD format                    │
│  Step 4:  Select Time Slot        → Choose from available                │
│  Step 5:  Enter Reason            → Describe visit purpose               │
│  Step 6:  Confirm Booking         → Press Y to confirm                   │
│  Step 7:  Sync to Calendar        → Press Y for Google Calendar          │
│                                                                           │
│  ⏱️  Average Time: 2 minutes                                             │
│                                                                           │
